Workshops

Here are some of the activities you can participate in during the first half of the day.

Python

Discover the powerful, dynamically typed, multi functional Python programming language

App Design

Learn how to create mobile applications using the Android Development Kit

Web Development

Learn PHP and JavaScript to create a beautiful website

Robotics

Utilize the powerful frameworks of QuickScope and apply the principles of acceptance test-driven development

Machine Learning

Calculate the gradient for a one layer, perceptron type neural network

CAD and 3D Printing

Bioinformatics

Inform yourself about bioinformatics using the R language

Frequently Asked Questions

What is AcademiesHacks?

A Hackathon at the Academies of Loudoun that educates students from age middle school to advanced high school various topics about computer science

When is AcademiesHacks?

May 18th from 9AM to 9PM.

Who can attend?

Students aged from middle school to advanced high school; the maximum age of participants is 19.

Is there any cost?

No, this event is fully funded thanks to our generous sponsors.

Is there a deadline to register?

Yes, the deadline to register is May 1st.

Is there a code of conduct?

Yes, please refer to this document

Do I need a team?

You can either register with a team, or we can assign you one.

What if I don't know how to code?

No worries! Our workshops are geared toward your skill.

What should I bring?

All you need to bring is yourself.
